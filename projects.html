Heart Failure Machine Learning

This project utilizes machine learning in order to perform retrospective analysis on patients admitted to the hospital for congestive heart failure. A model is developed to predict hospital readmittance or mortality. The application is coded in python and makes use of XGBoost and various other python tools for data storage and analysis. The model is used in medical papers and hospital practice.

Aircopter Project

This project was done in association with the IEEE student branch at UCLA. The main goal is to design a drone in both hardware and software from scratch. This includes designing and assembling a PCB, writing high and low level code with simple libraries and inputs, and understanding the relationship between hardware and software. The software was primarily written in C++ and the hardware was designed through EagleCAD.

Magic Mirror

The magic mirror was a summer project that involved creating a user interface application using a raspberry pi and a spare monitor. The project focused on the connection between sensors and the digital world in an IoT application by using sensor readings to display information like weather and adjust smart home systems. Additionally, support was added for smart devices like a Nest Thermostat as well as voice assistant application like Alexa and Google Assistant.
